## Quick review

The third version of the aiogram is very interesting. There are many changes here, the most important of them: [routers](https://docs.aiogram.dev/en/dev-3.x/dispatcher/router.html), [callback factory](https://docs.aiogram.dev/en/dev-3.x/dispatcher/filters/callback_data.html#usage), [magic filters](https://docs.aiogram.dev/en/dev-3.x/dispatcher/filters/magic_filters.html). Since I will have to switch to the third version in the future, I decided to make a template for it now and use it. In some files you will find links to examples.
